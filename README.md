# support-ticket-classification

## Опис
Telegram-бот для класифікації звернень користувачів (технічні, фінансові, інформаційні) з базовою статистикою моделі та логуванням дій.

## Реалізовані функції
- Прийом повідомлень користувача та автоматична класифікація тексту за 3 категоріями.
- Команди бота: старт, підказка, відправлення звернення, перегляд статистики.
- Відповідь на некоректні повідомлення (фото, файли, стікери тощо).
- Обробка помилок Telegram API та мережевих збоїв.
- Навчання Naive Bayes класифікатора на старті додатка.
- Розбиття даних на train/test із фіксованим seed для відтворюваності.
- Розрахунок метрик (accuracy, correct/total) для тестової вибірки.
- Логування дій користувача та системних подій у файли.
- Налаштування через змінні середовища (BOT_TOKEN) та JSON-конфіг.

## Основні модулі
- Бот та обробники команд/повідомлень: `module_1/src/bot`.
- Модель класифікації та навчальні дані: `module_1/src/nlp`.
- Утиліти (клавіатура, логер, shuffle): `module_1/src/utils`.
- Конфігурація: `module_1/src/configs`.
